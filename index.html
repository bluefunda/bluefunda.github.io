Get Started

# Easiest: log in via browser
open https://bluefunda.com/login
# Or install the CLI from source
go install github.com/bluefunda/bluefunda-ai/cmd/bai@latest
# Authenticate via OAuth device flow, then start chatting
bai login
bai chat start "Summarize last quarter's sales from SAP"

BlueRequests

Event-driven change and release management for SAP operations — manage transport requests, change orders, approvals, and release workflows from the command line or a TUI dashboard.

bluerequests →

CLI for BlueRequests change and release management (breq binary) — TUI dashboard, CRs, events, approvals.

Quick Install

# Homebrew (macOS/Linux)
brew tap bluefunda/tap
brew install --cask breq
# One-line installer
curl -fsSL https://raw.githubusercontent.com/bluefunda/bluerequests/main/install.sh | sh
# Or install from source
go install github.com/bluefunda/bluerequests/cmd/breq@latest
# Authenticate and open the TUI dashboard
breq login
